开始学习markdown

发表于 2016-10-31

Markdown 是一种轻量级的「标记语言」，它的优点很多，用起来很方便，也越来越受欢迎，所以也打算学习一下，经过了解确实是很简单而且很实用。下面总结一下常用的几个标记。

标题

标题写法是# 标题名，几级标题加几个#，示例： 

# 一级标题  
## 二级标题  
...

列表

列表的写法是在文字前加上短横线-，示例：

  • 文本1
  • 文本2
  • 文本3

有序列表是在文本前加上数字，示例：

  1. 文本1
  2. 文本2
  3. 文本3

加粗和斜体

加粗是**文本**，斜体是*文本*，示例：
加粗
斜体

链接和图片

链接的写法[文本](链接地址)，示例：
我的博客
图片的写法跟链接很相似，只是在前面加一个！，![](链接地址)，示例：

引用和引用代码

引用的写法是在引用文本前加>，示例：

孤帆远影碧空尽
唯见长江天际流

引用代码是将代码放到``中，示例：
hello world
